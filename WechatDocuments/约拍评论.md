# 约拍评论

url: weixin/appointment/comment

| code  | 值           |
| ----- | ----------- |
| 200   | 评论成功        |
| 500   | 数据库提交错误     |
| 40001 | 无评论内容       |
| 40002 | 该用户未参加此次约拍  |
| 40003 | 无评分         |
| 40004 | 该约拍项不存在或已失效 |
| 40005 | 该用户不存在      |

### 请求参数：

|       |      |          |
| ----- | ---- | -------- |
| apid  |      | 约拍的Id    |
| utel  |      | 评论用户的手机号 |
| score |      | 评分       |
