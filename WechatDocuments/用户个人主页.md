# 用户个人主页

url：域名/weixin/userpage/selfinfo

| code  | 说明              |
| ----- | --------------- |
| 200   | 成功              |
| 40001 | 该用户不存在          |
| 40002 | 该用户作为摄影师没有发布过约拍 |
| 40003 | 该用户作为模特没有发布过约拍  |
| 40004 |                 |
| 40005 | 获取评论用户出错        |



| type值 | 请求           |
| ----- | ------------ |
| 1     | 请求用户自己的个人主页  |
| 2     | 请求看其他用户的个人主页 |



### 1.  看自己的个人主页：

返回评论、用户自我介绍

### 请求规范：

| 值               | 类型及实例 | 描述          |
| --------------- | ----- | ----------- |
| type            | 1     | 类型          |
| # openid（前期不用发） | 7     | 用户个人的openid |
| utel            |       | 加密后的手机号     |

@attention:获取用户自我介绍及评论时请判断是否为空

| 返回值      | 说明       |
| -------- | -------- |
| code     | 状态码      |
| contents | 返回说明     |
| comment  | 评论内容     |
| alais    | 评论的用户的昵称 |
| score    | 获得的评分    |
| title    | 此次约拍的名称  |
