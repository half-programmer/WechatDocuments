# 删除约拍

url： weixin/appointment/delete

| Code  | 说明              |
| ----- | --------------- |
| 40001 | 该用户不存在          |
| 40002 | 该约拍不存在          |
| 40003 | 该约拍已被删除         |
| 40004 | 该约拍已有人报名，不能删除   |
| 40005 | 只有报名阶段的约拍才可以被删除 |
| 500   | 数据库提交失败         |
| 200   | 删除成功            |
